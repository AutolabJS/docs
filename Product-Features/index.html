    Product Features

    A summary of project features are:

    1. Auto-evaluation support for programming labs.
    2. Support for multiple languages. The languages supported are: C, C++, Java, Python.
    3. Automated score board listing maximum marks for each of students.
    4. Uses Gitlab to provide git version control to students.
    5. Uses Docker containers to separate the Autolab components.
    6. Support for unit / integration / functional and Input-Output testing strategies.
    7. Ansible-based auto configuration and installation procedure.
    8. On demand auto re-evaluation for instructors. (this feature needs bug fixing)
